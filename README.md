IMPORTANT: Logo Display Instructions
=====================================

For the logo to display properly, you MUST have BOTH files in the same folder:

1. index.html
2. Sereno_Logo_V1.png

These files must be in the exact same directory/folder.

If you only download index.html, the logo will not appear.
If the files are in different folders, the logo will not appear.

SOLUTION:
---------
1. Download BOTH files
2. Put them in the same folder
3. Open index.html in your browser

The logo will then display correctly in the header and footer.
